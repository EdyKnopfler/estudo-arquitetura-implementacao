# Estudo em arquitetura: implementação

Implementação do projeto planejado em https://github.com/EdyKnopfler/architecture-studies/.

## Construir e rodar

Imagem do backend das reservas:

```bash
cd reservas-backend
./build-image.sh
cd ..
```

Subindo os contêineres:

```bash
docker compose up [-d]
```
